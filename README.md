# Learning Dashboard

A simple, responsive Learning Dashboard built using **HTML**, **CSS**, and **JavaScript**.  
This project was created as part of the **Next Chapter Technical Pre‑Course Assessment** to demonstrate core web development skills without using external libraries or frameworks.

---

## 🚀 Live Demo
View the project here:  
https://jkbrooksdev.github.io/nextchapter-precourse-site

---

## 📘 About the Project
The Learning Dashboard is a clean, beginner‑friendly webpage designed to showcase:

- Semantic HTML structure  
- Responsive CSS layout  
- Functional JavaScript logic  
- Lists, tables, forms, and media elements  
- Proper use of attributes and special characters  

The site includes multiple sections such as an About area, study resources, a JavaScript‑powered goal tracker, and embedded audio/video learning materials.

---

## 🛠️ Technologies Used
- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla)**  
- **Git & GitHub Pages** for hosting

No external libraries or frameworks were used, per project requirements.

---

## 📂 Features

### ✔ Semantic HTML Structure
Includes `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`.

### ✔ Responsive Design
Scales smoothly across desktop, tablet, and mobile screen sizes.

### ✔ Study Resources Section
Contains:
- Unordered list  
- Ordered list  
- A fully styled table  

### ✔ Interactive JavaScript Feature
A **Study Goal Tracker** that allows users to:
- Add new goals  
- Display them dynamically  
- Update a running total  
- Prevent empty submissions  

### ✔ Media Integration
- Embedded **audio** (online MP3 source)  
- Embedded **video** (YouTube iframe)  

### ✔ Special Characters
Includes: `&copy;`, `&nbsp;`, `&lt;`, `&gt;`, and `&amp;`.
